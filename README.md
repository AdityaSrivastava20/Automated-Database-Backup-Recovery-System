# Automated Database Backup & Recovery System

## Overview

This project automates PostgreSQL database backup and recovery using Python and shell scripting.

## Features

- Automated PostgreSQL backups
- Timestamped backup files
- Backup logging
- Database recovery
- Scheduled cron jobs
- Old backup cleanup

## Technologies Used

- Python
- PostgreSQL
- Linux
- Shell Scripting
